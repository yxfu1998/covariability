# Monte Carlo Covariability Analysis of Tropical Moisture and Temperature Trends

**Authors:** Yixiao Fu, Cheng-Zhi Zou, Yu Wang, Chunyang Li, and Shi Liu

This repository provides the data and code used to analyze the covariability between tropical moisture and temperature trends using a Monte Carlo sampling approach. The associated manuscript is entitled:

**“Revisiting the Covariability between Tropical Moisture and Temperature Trends Using a Monte Carlo Sampling Approach”**

The purpose of this repository is to facilitate reproducibility of the analyses presented in the manuscript. The original climate datasets used in this study are described in the Supplementary Information of the paper. For convenience, the processed datasets required for reproducing the main analyses are provided in the data directory.
